# @aviato-media/library-movies

Movie library with rich metadata display, filtering, and sorting. Bundles per-folder, with built-in classification for trailers, behind-the-scenes, deleted scenes, featurettes, and other extras.

## Capabilities

- `library` — defines the `movies` library type.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `library-movies@<version>`).
