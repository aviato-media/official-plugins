# @aviato-media/audible

Indexes audiobook files by resolving metadata from [Audible](https://www.audible.com/) via the [audnex.us](https://audnex.us/) API. No Audible API key required.

## Capabilities

- `indexer` — matches audiobook files to Audible entries.

## Configuration

- `region` (select, default `us`) — Audible storefront. ASIN lookups use this region as a hint. Choices: `us`, `ca`, `uk`, `au`, `fr`, `de`, `jp`, `it`, `in`, `es`.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `audible@<version>`).
