# @aviato-media/library-audiobooks

Audiobook library with author/narrator/series browsing and chapter display. Bundles `.m4b` per-file, other formats per-folder; companion `.pdf` / `.opf` / cover-art picked up automatically.

## Capabilities

- `library` — defines the `audiobooks` library type (panel queue, eager strategy, autoplay).

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `library-audiobooks@<version>`).
