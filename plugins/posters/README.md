# @aviato-media/posters

Pipeline subscriber that discovers local poster, fanart, banner, thumb, landscape, and cover image files alongside media — the SeeFood of your library.

## Capabilities

None. Subscriber-only.

## Subscriptions

- `pipeline.probe.afterProcess` — finds local artwork files and contributes them as artwork references on the bundle.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `posters@<version>`).
