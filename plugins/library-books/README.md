# @aviato-media/library-books

Ebook library with author, series, and publisher browsing. Bundles per-file across `.epub`, `.pdf`, `.doc`, `.docx`, `.txt`. Non-seekable, no idle-protection.

## Capabilities

- `library` — defines the `books` library type.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `library-books@<version>`).
