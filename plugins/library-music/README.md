# @aviato-media/library-music

Music library with artist / album / track browsing, audio tag display, and filtering.

## Capabilities

- `library` — defines the `music` library type.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `library-music@<version>`).
