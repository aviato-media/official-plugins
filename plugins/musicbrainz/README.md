# @aviato-media/musicbrainz

Indexes music files by resolving metadata from [MusicBrainz](https://musicbrainz.org/) and cover art from the [Cover Art Archive](https://coverartarchive.org/).

## Capabilities

- `indexer` — matches music files to MusicBrainz recordings / releases.

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `musicbrainz@<version>`).
