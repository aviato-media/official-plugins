# @aviato-media/library-tv

TV show library with series/season/episode browsing, rich metadata display, and filtering. Bundles per-file with `SxxExx`-based identity scoped to the parent directory.

## Capabilities

- `library` — defines the `tv` library type (episodic, with up-next queue).

## Installation

Install via the Aviato in-app marketplace, or download the latest tarball from the [GitHub Releases](https://github.com/aviato-media/official-plugins/releases) page (tag `library-tv@<version>`).
